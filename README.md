# RestFlow

**RestFlow** is a lightweight, minimal web based REST client built on **Sveltekit**, inspired by tools like Thunder Client and Postman â€” but focused on speed, simplicity, and zero bloat.

Perfect for quick API testing without heavy Electron apps.

---

## Features

- Super lightweight & fast
- Supports common HTTP methods
  - `GET`, `POST`, `PUT`, `DELETE`
- Request body support
  - Raw JSON / Text
  - Form Data (multipart)
- Reactive state using Svelte stores
- Response viewer
  - Status code
  - Headers
  - Raw response body
- Clean, minimal UI

---

## Built With

- **Sveltekit**
- Native **Fetch API**
- Svelte Stores
- Pure CSS (no UI frameworks)
